# 🚀 Amazon Customer360 Analytics Platform

## 📌 Project Overview

The **Amazon Customer360 Analytics Platform** is an end-to-end Azure Data Engineering project designed to integrate customer information from multiple business systems into a unified analytics platform.

This project follows the **Medallion Architecture (Bronze → Silver → Gold)** to transform raw business data into analytics-ready datasets using Azure cloud services.

The final outcome is a **Customer360 Gold Layer**, providing a complete 360-degree customer view for reporting, analytics, and business decision-making.

---

# 🎯 Business Objective

Organizations often store customer information across multiple independent systems.

This project combines data from:

- 👤 CRM
- 🛒 Transactions
- 📦 Products
- 🎫 Support Tickets
- 📢 Marketing Campaigns

to create a **single source of truth** for every customer.

---

# ⚡ Project Goals

- ✅ Build an end-to-end Azure ETL pipeline
- ✅ Implement Medallion Architecture
- ✅ Clean and standardize raw datasets
- ✅ Generate a Customer360 Gold Layer
- ✅ Automate data transformation using Azure Data Factory
- ✅ Store business-ready Delta tables
